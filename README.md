# 알림펫 
> 반려동물을 위한 알림 서비스

~~~
우리의 반려동물을 체계적으로 관리하며, 그들의 건강과 행복을 책임지는 편리한 알림 서비스

반려동물의 케어를 체계적으로 관리하는 서비스로, 밥 시간, 물 갈아주기 등의 일상적인 케어뿐만 아니라
특별한 기념일까지도 알림을 통해 손쉽게 관리할 수 있습니다.

개인 맞춤형 서비스를 통해 반려동물의 건강과 행복을 놓치지 않고 지켜보세요!
~~~
